Linux Commands

cd [directory]

  • changes the current directory
    • if no directory name is entered, it will change to the user's home directory
  • example of usage: a user wanting to change from their home directory to their directory titled projects would use "cd projects"

ls [directory]

  • lists the contents of the directory
    • if no directory name is entered, it will list the contents of the current directory that the user is in
  • example of usage: a user wants to delete a directory but decides to check to see if the directory is empty first; they would use "ls [directory]" to check

mkdir [directory]

  • creates a new directory
    • note: [directory] parameter refers to the name of the new directory that the user desires to create
  • example of usage: a user wanting to create a new directory entitled "CSP" would use the command "mkdir CSP"

cat [file]

  • used to display a file
    • note: [file] parameter refers to the name of the new file that the user wishes to display
  • example of usage: a user wanting to display a variable she initiated would use "cat [variable name]"
